Recursive

Recursive terjadi ketika function memanggil dirinya sendiri secara langsung atau tidak langsung untuk menyelesaikan sebagian masalah yang diberikian.

Struktur Recursion

Ada 2 bagian penting:
1. Base Case: Bagian dari function yang berfungsi untuk menghentikan recursion. kondisi dimana masalah tidak lagi membutuhkan pemanggilan function secara recursive.
2. Recursive Case: Function akan memecah masalah menjadi sub-masalah yang lebih kecil dan memanggil dirinya sendiri.
